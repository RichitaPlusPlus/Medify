# Medify

A web/mobile app designed to automate scheduling for consulting services and specialized clinics, enabling users to plan and request medical appointments

It will operate in a modular and independent manner to be used in clinics or hospitals, thereby automating the scheduling process and maximizing cost efficiency by reducing the need for a secretary to just an appointment manager within the app




## Features

- multiple languages
- Light/dark mode toggle
- Live previews
- image uploader
- Cross platform


## FAQ

#### Will it be available nationally or internationally?

-Currently, it is only available nationally, but negotiations will take place at the regional level in the future.

#### Is it for general medicine, or just for specific conditions?

It’s actually for both; we have various specialists, ranging from pediatricians to neurosurgeons.

#### Could an app based on this be developed for each clinic/hospital?

Yes, that’s the main idea—that each practice can have its own specific Medify.
